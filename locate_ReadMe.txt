# It is a simple implementation of locate tool (available in Unix) for Windows


To run this tool , save it to some location
	a) Open a cmd shell
	b) Change directory to location where you have saved locate.py file
	c) Run the file by writing
			
			python locate.py filename FAST
	d) Enter the directory name where to search
	
# To see fast results search in accurate directories.
